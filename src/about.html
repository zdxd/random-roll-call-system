关于此工具：

此工具由ZD XIAODA（张思源）制作，底层采用HTML+CSS3+JS编写，Windows端采用NW.JS打包。真诚的感谢所有使用到的开源项目。

使用场景：

公司年会抽奖，老师上课点名，派对活跃气氛。。。

版本区别：

Web版需要部署在服务器使用，名单只能服务器管理员修改，适合同一个名单但是多设备使用的用户。（Web版支持Firefox，Microsoft Edge，Google Chrome，Safari等所有主流浏览器）

Windows需要Windows 7及以上的版本才能使用，适合同一个设备同一个名单的用户。（如班级点名）

使用到的项目：

mdui：一个基于 Material Design 的前端框架。

requirejs：A file and module loader for JavaScript

nw.js：Call all Node.js modules directly from DOM/WebWorker and enable a new way of writing applications with all Web technologies.

NSIS：一个开放源代码脚本驱动的封装安装档用工具。

iconv-lite：Pure JS character encoding conversion

非常感谢！
